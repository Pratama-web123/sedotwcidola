# Sedot WC Idola

## Deskripsi
Website layanan Sedot WC Idola yang dirancang untuk menampilkan informasi layanan secara jelas, responsif, dan mudah diakses oleh pengguna di berbagai perangkat.

## Fitur Utama
- Responsive UI  
  Tampilan website sudah responsif dan optimal di berbagai perangkat, termasuk smartphone, tablet, dan desktop.

- Optimasi SEO
  Struktur HTML dan elemen website telah dioptimalkan untuk meningkatkan visibilitas di mesin pencari.

- Integrasi WhatsApp (API) 
  Tersedia tombol WhatsApp yang langsung mengarahkan pengguna ke kontak layanan dengan pesan siap kirim (click to chat).

- Lokasi Layanan
  Menyediakan informasi area layanan:
  - Purwokerto  
  - Purbalingga  
  - Banjarnegara  
  - Cilacap  

- **Integrasi Google Maps**  
  Menampilkan lokasi layanan melalui peta interaktif untuk memudahkan pengguna menemukan lokasi.

## Teknologi
- HTML
- CSS
- JavaScript

## Tujuan Pengembangan
Membuat website company profile layanan sedot WC yang informatif, mudah digunakan, serta mendukung peningkatan jangkauan pelanggan melalui optimasi digital.
